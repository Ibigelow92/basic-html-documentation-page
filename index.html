HTML element
(Written as <HTML>)

The HTML element is your root element. This means it is the element from which all the other ones descend.

Head element
(Written as <head>)

The head element contains the metadata about your document. This is the machine-readable data. Some examples of metadata are the title, scripts and style sheets.

Meta element
(Written as <meta>)

Like the head element, the meta element represents a collection of metadata. Specifically though, it defines the metadata that cannot be represented by other HTML meta-related elements.

Title element
(Written as <title>)

The title element defines how your document will be titled in a browser's toolbar or tab.

Link element
(Written as <link>)

A link element connects your document to an external resource.

Body element
(Written as <body>)

The body element defines the content in a document, which is readable by the person viewing it in a browser.

Header element
(Written as <header>)

The header element contains a document's introductory content. This could be a line of navigational aids or it could include things like a logo or a search form.

Main element
(Written as <main>)

The main element defines the primary content of a document. This content directly relates to the topic or functionality of the document.

Heading elements
(Written as <H1>-<H6>)

The elements H1-H6 represent a set of text headings of varying degrees of importance, with H1 at the top and H6 at the bottom. Headings that are of a lower number will appear with more visual emphasis and vice versa.

Nav element
(Written as <nav>)

The nav element defines a clickable navigation link. This link may direct the user to another document or to another place within the same document.

Div element
(Written as <div>)

The div element creates a container around flow content. It does not create a visually observable change to the content until styled in CSS.

List element
(Written as <li>)

As its name suggests, the list element creates a list of items.

Ordered list element
(Written as <ol>)

The ordered list element defines a list that is ordered in some way, typically numbered.

Unordered list element
(Written as <nav>)

The unordered list element defines a list that is not ordered.
